\section{Exercise : $b$-flow}

The solution to case (a) can be seen in figure \ref{fig:ex1a}. Case (b) has no solution. In figure \ref{fig:ex1b}, which is a reproduction of figure 1b from the assignment, we can see that $v4$ has a demand of $-2$, but no outgoing edges -- it can never get rid of any flow, but it needs to get rid of two more units than it gets in.
